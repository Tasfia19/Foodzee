# Project Title-Foodze
Foodze is a food delivery website,where a user can order food  and a admin panel where the admin can add,list  & change track of ordered foods.I have used MERN stack for this project.


# How to run the project
# Frontend
cd frontend
npm install //it will install all necessary dependencies 
npm run dev

# Admin Panel
# Admin
cd admin
npm install //it will install all necessary dependencies 
npm run dev

# Backend

cd backend
npm install //it will install all necessary dependencies 
npm start

# Database
MongoDB configure it in config --->  db.js
